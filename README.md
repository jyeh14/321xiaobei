# 小北門後巷計畫｜Xiao Beimen Lane Project

台南市北區巷弄聚落再生設計紀錄網站。

## 檔案結構

```
├── index.html           # 0號筆記（主頁）
├── site-analysis.html   # 基地分析
├── inspiration.html     # 建築靈感
├── kyoto-machiya.html   # 京都町屋研究
├── architect-notes.html # 建築師討論紀錄
├── sketches.html        # 設計草圖
├── construction-log.html# 施工日誌
├── future-vision.html   # 未來生活構想
└── style.css            # 共用樣式
```

## 使用方式

直接在瀏覽器中開啟 `index.html`，或上傳至 GitHub Pages 部署。

### GitHub Pages 部署步驟
1. 建立 Repository（Public）
2. 上傳所有檔案至根目錄
3. Settings → Pages → Source 選擇 `main` branch
4. 等待約 1 分鐘，即可透過 `https://[username].github.io/[repo-name]` 存取

## 技術規格

- 純 HTML + CSS，無需任何框架或編譯步驟
- 所有圖示為 SVG（inline），無外部圖片依賴
- 字型透過 Google Fonts CDN 載入（Noto Serif TC、Noto Sans TC）
- RWD 支援，適配手機與桌面瀏覽器
